6. SETUP AND COMPILATION

6.1 Prerequisites

Required:
- Java Development Kit (JDK) 22 or higher
- SQLite JDBC driver (sqlite-jdbc-3.x.x.jar)

Optional (for GUI):
- JavaFX SDK 22 or compatible version

6.2 Compilation Instructions

Windows:
1. Open Command Prompt
2. Navigate to project directory
3. Run: build.bat
4. Check bin directory for .class files

Linux/Mac:
1. Open Terminal
2. Navigate to project directory
3. Run: chmod +x build.sh && ./build.sh
4. Check bin directory for .class files

Manual Compilation:
javac -d bin src/*.java src/database/*.java src/models/*.java src/services/*.java src/ui/*.java src/utils/*.java

6.3 Running the Application

Console Application:
java -cp bin;lib/sqlite-jdbc-3.x.x.jar LibraryConsoleApp

GUI Application:
java -cp bin;lib/sqlite-jdbc-3.x.x.jar --module-path /path/to/javafx/lib --add-modules javafx.controls ui.LibraryManagementGUI

6.4 Database Setup

Automatic Setup:
- Database creates automatically on first run
- Tables initialize if not present
- Sample data loads for testing

Manual Database Reset:
- Delete library_system.db file
- Restart application
- Database recreates with sample data
